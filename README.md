# Board 
## 소프트웨어 시스템 개발 복습 프로젝트
### 개요
- 프로젝트 기간 : 2022년 7월 1일 ~ 8월 23일
- 사용 기술 : **Spring, Bootstrap**
---
### 프로젝트 소개
_**‘ 스프링부트를 사용한 웹 서비스 개발 ’**_ 
---
### 주요 기능
1. SpringSecurity를 사용한 로그인
2. 게시글
- 게시글 작성 / 수정 / 삭제 / 읽기
- 게시글 내 댓글 / 좋아요 / 추천 / 검색
- 게시글 페이징네이션
- 게시글 내 이미지 첨부
3. 상품 판매
- 장바구니 상품 추가 / 삭제 / 금액 계산
- 상품 리뷰 작성 / 수정
4. 관리자 페이지
- 회원 프로필 페이지
- 회원 탈퇴
### 소감
4학년 1학기 '소프트웨어 시스템 개발' 강의 중 배웠던 내용들을 복습하며, 개념을 정리했던 프로젝트였다.
